# Яндекс.Музыка — глобальные хоткеи

Chrome/Edge расширение для управления Яндекс.Музыкой с **любой вкладки**.

## Горячие клавиши

| Клавиша | Действие |
|---|---|
| `Alt+P` | ▶/⏸ Пауза / Воспроизведение |
| `Alt+→` | ⏭ Следующий трек |
| `Alt+←` | ⏮ Предыдущий трек |
| `Alt+↑` | 🔊 Громче (+10%) |
| `Alt+↓` | 🔉 Тише (−10%) |
| `Alt+L` | ❤️ Лайк |

Клавиши можно переназначить через `chrome://extensions/shortcuts`.

## Установка

1. Скачай репозиторий (Code → Download ZIP) и распакуй
2. Открой `chrome://extensions/` → включи **Режим разработчика**
3. Нажми **«Загрузить распакованное»** → выбери папку

## Как работает

- `background.js` — перехватывает глобальные горячие клавиши через `chrome.commands` API
- `content.js` — инжектируется на страницу `music.yandex.ru` и управляет плеером через DOM
- При нажатии появляется тост-уведомление с действием и уровнем громкости
